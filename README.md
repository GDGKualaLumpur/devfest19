# DevFest 2019 Kuala Lumpur PWA

Progressive Web App for Google DevFest 2019 Kuala Lumpur.

![](https://i.imgur.com/QCyPvdX.jpg)

## Getting Started

Start development server:

`npm run dev`

Build:

`npm run build`

Deploy:

`firebase deploy`

## Technology Stack

* Preact
* Firebase (Auth, Realtime Database, Hosting)
* Service Worker
